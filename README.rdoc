= openssl-pkcs8

This adds PKCS8 compliant key export to the OpenSSL::PKey::RSA library that
is part of standard Ruby distributions.

Example:

  key = OpenSSL::PKey::RSA.new(1024)
  
  key.to_pem_pkcs8
  # => "-----BEGIN PRIVATE KEY----- ..."
